## 项目介绍
该项目基于 SpringBoot 2.x + Vue Element UI 实现，该项目目前是一个信访系统，有用户、事业单位员工和领导三种业务角色。
大家可以根据自己的需求随便改成其他系统。

## 技术组成
- Vue2.x
- SpringBoot 2.x
- MyBatis / MyBatis Plus
- MySQL
